# Personal Portfolio Website

A responsive and modern Personal Portfolio Website developed using HTML, CSS, and JavaScript. The website is designed to showcase my profile, technical skills, projects, and contact information in a professional way.

* Responsive Navigation Bar
* Hero Section
* About Me Section
* Skills Showcase
* Projects Gallery
* Contact Form
* Interactive User Interface
* Mobile-Friendly Design

## Technologies Used

* JavaScript
